# 📘 E-Commerce Web Scraper and Flask Dashboard

This project demonstrates a **web scraping application** that extracts product data from a dynamic e-commerce website and displays it through a **Flask web server** with an HTML table. It's a beginner-friendly yet complete project for practicing both **Python web scraping** and **Flask integration**.

---

## 🔧 Technologies Used

- Python 3.x  
- BeautifulSoup 4 (`bs4`)  
- Requests  
- Flask  
- HTML & CSS (Jinja2 templating)

---

## 🌐 Website Scraped

**URL**: [https://books.toscrape.com/](https://books.toscrape.com/)

This demo site provides product data like:
- Book Title  
- Price  
- Rating

---
